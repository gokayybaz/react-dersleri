React Component Kavramı
-------------------------

Temel olarak component'leri şu şekilde düşünebiliriz;

Lego gibi : küçük parçalar birleşerek -> büyük parçaları oluşturuyor -> büyük parçalarda yapıyı oluşturuyor.

Örnek olarak : NavBar bir büyük parçadır; title'ı küçük parçadır, button'u küçük parçadır, logosu küçük bir parçadır.
               Bunlar birleşince NavBar'ı oluşturmaktadır. NavBar, Body, SideMenu, Footer'da birleşince büyük yapıyı oluşturmaktadır.

Temel olarak component mantığı bu şekildedir, Özellikle kendini tekrar eden yani tekrar tekrar kullanılacak alanlar için büyük bir
kolaylık sağlamaktadır.

React'da Component'ler iki çeşittir;

1-> Class Component
2-> Function Component

Bu iki component hakkında farklılıklar, detaylı bilgileri ilerleyen notlarımda paylaşacağım.
