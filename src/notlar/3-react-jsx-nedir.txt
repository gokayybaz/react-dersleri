JSX Nedir?

JSX React tarafından geliştirilen bir biçimlendirme dilidir, Temel amacı Javascript ile HTML'i iç içe beraber yazmamızı sağlamaktır.

JSX içerisinde Javascript kodları : {console.log("Hello World")} şeklinde süslü parantezler içerisinde yazılır.

Komponentler temel olarak javascript kodlarıdır, fakat bize JSX türünde bir HTML tagları barındıran bir yapı dönmektedirler.

Dönen JSX'ide React'ın VirtualDOM yapısı işleyip, normal html kodlarına ve içerisinde javascript var ise javascript'e çevirip index.html'e gönderir.

Tabi normal HTML ile JSX kodları arasında bazı attribute'larda farklılık vardır bunlardan bazıları;

// HTML : <input type="text" class="my-input"> | JSX : <input type="text" className="my-input">

// HTML : <input type="checkbox" checked> | JSX : <input type="checkbox" defaultChecked>

// HTML : <label for="inp-userName">Kullanıcı Adı</label> | JSX : <label htmlFor="inp-userName">Kullanıcı Adı</label>

// HTML : <div style="display:none;">Mobile Menu</div> | JSX : <div style={{display:none}}>Mobile Menu</div>
                                                         Style'ları süslü parantezler içerisinde bir obje oluşturup veriyoruz.
                                                         Style isimleri camelCase olarak belirtilip değerleri ise "" içerisinde belirtilmelidir.

// HTML : <input type="text" value="Deneme"/> | JSX : <input type="text" defaultValue="Deneme"/>

Not : Bir Component içerisinde bir tane JSX Kapsayıcı/Genel etiket bulunmak zorundadır.
